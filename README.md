# Page Baccarat Resource

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面，主要服务于展示、测试或参考用途。页面内容不针对任何特定域名或网站，仅为通用资源集合。

## 目录说明

```
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面所需的静态资源（如 CSS、JS、图片等）
└── README.md       # 本说明文件
```

- **pages/**：每个 HTML 文件为一个独立页面，文件名即页面标识。
- **assets/**：按需存放页面引用的资源文件，建议按功能或页面分组。

## 页面归档说明

- 所有页面均为独立 HTML，可直接在浏览器中打开。
- 页面之间无强依赖关系，但可能共享部分公共资源（如通用样式、脚本）。
- 归档遵循“一页一文件”原则，便于管理和引用。

## 维护说明

- 新增页面：将 HTML 文件放入 `pages/` 目录，如有资源请对应放入 `assets/`。
- 更新页面：直接修改对应 HTML 文件，注意保持资源路径一致。
- 删除页面：移除对应文件及关联资源。
- 提交规范：请附带简短说明，如“新增页面：xxx”或“更新页面：xxx”。

## 注意事项

- 本仓库不提供在线演示或部署服务，仅作为文件归档。
- 页面内容仅供学习、测试或参考，请勿用于其他用途。
- 如有问题或建议，欢迎提交 Issue 或 Pull Request。
